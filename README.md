# TopoAcoustic Discovery Engine
TopoAcoustic is a standalone music recommendation platform that models audio tracks as continuous geometric manifolds. By using Topological Data Analysis rather than behavioral collaborative filtering, the system maps the intrinsic acoustic topology of music to surface long-tail and niche artists without relying on historical listener data.

## The Problem
Traditional recommendation engines utilize collaborative filtering, which depends on massive user-item interaction matrices. This creates a systemic "cold-start" problem: if an independent artist has very few streams, the algorithm cannot calculate a reliable latent vector. Consequently, the true long-tail of music remains mathematically isolated from mainstream discovery.

## The Solution
This platform completely bypasses human interaction data and popularity metrics. Instead, it extracts high-dimensional acoustic features directly from the raw audio signal.

Using Takens' embedding theorem and Vietoris-Rips filtrations, the engine calculates the persistent homology of the track. The resulting persistence landscapes are indexed in a high-dimensional space. Recommendations are generated by querying this metric space for tracks that share the exact same topological invariants, allowing a track with zero listeners to be surfaced alongside mainstream hits based purely on structural similarity.

## Core Architecture
- Frontend Layer: React, TypeScript, Tailwind CSS, WebGL (Manifold Visualization)
- API & Orchestration: FastAPI, Python, Supabase
- Topological Engine: librosa (Signal Processing), giotto-tda & gudhi (Persistent Homology), pot (Optimal Transport)
- Indexing: hnswlib (Approximate Nearest Neighbor Vector Search)
